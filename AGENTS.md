# Study Buddy AI — Project Rules

## Tech Stack
- Python 3
- FastAPI
- OpenAI API
- python-dotenv

## Code Style
- Use type hints
- Add docstrings to functions
- Add try/except error handling

## API Response Format
All endpoints return:

{
  "status": "ok",
  "data": ...
}

## Project Structure
- Keep it simple
- Everything in main.py

## Secrets
- API keys stored in .env
- Never hardcode API keys
