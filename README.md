# city-expert
this lovely app will help you plan your day in the city.
Just tell it when, where and for what and enjoy this easy extension!

Instalation: add a .env file in your folder (preferabbly do this in visual studio)

Include these in your .env file
OPENROUTER_API_KEY=your key here 
OPENROUTER_BASE_URL=https://openrouter.ai/api/v1
OPENROUTER_MODEL=openai/gpt-4o
OPENROUTER_APP_TITLE=City ActivityAdvisor

and have fun!
